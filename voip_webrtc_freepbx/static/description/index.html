VoIP WebRTC FreePBX

Full-featured VoIP and WebRTC integration for Odoo 17

Overview

This module provides comprehensive VoIP functionality for Odoo 17 Community Edition, enabling seamless integration with FreePBX servers using WebRTC technology. Make and receive calls directly from your Odoo interface with full call management capabilities.

Key Features

WebRTC Calling

Browser-based calling without plugins using modern WebRTC technology

FreePBX Integration

Direct connection to your FreePBX server via SIP/WebSocket

Call Recording

Automatic call recording with playback and download capabilities

Call History

Detailed call logs with duration, status, and contact association

Contact Integration

Automatic contact matching by phone number

Recording Sharing

Share call recordings with other users in the system

Multi-language

Full support for Arabic and English languages

Clean Code

Built with clean architecture and best practices

Installation

  1. Download the module and place it in your Odoo addons directory
  2. Update the apps list in Odoo
  3. Install the "VoIP WebRTC FreePBX" module
  4. Download SIP.js library from GitHub and place it in static/lib/sip.js
  5. Configure your FreePBX server connection
  6. Set up VoIP users with their SIP credentials

Configuration

VoIP Server Setup

  1. Go to VoIP → Configuration → VoIP Servers
  2. Create a new server with your FreePBX details:
    • Server Name: A friendly name for your server
    • Host: Your FreePBX server hostname or IP
    • WebSocket URL: WebSocket URL (e.g., wss://your-server:8089/ws)
    • STUN Server: Google's public STUN server is configured by default
  3. Test the connection

VoIP User Configuration

  1. Go to VoIP → Configuration → VoIP Users
  2. Create a new user configuration:
    • Odoo User: Select the user
    • VoIP Server: Select configured server
    • SIP Username: Extension number from FreePBX
    • SIP Password: SIP password from FreePBX

Usage

Making Calls

  1. Click the phone icon in the systray
  2. Enter the phone number or use the dialpad
  3. Click "Call" to initiate the call

Call History

Access your call history from VoIP → Calls → Call History

Call Recordings

View and manage recordings from VoIP → Calls → Recordings

Share recordings with other users using the "Share" button

Technical Details

  • WebRTC Library: SIP.js (open source)
  • Protocol: SIP over WebSocket
  • NAT Traversal: STUN/TURN support
  • Audio Codec: Opus (WebM)
  • Security: TLS/WSS encryption support
Note: This module requires a properly configured FreePBX server with WebRTC support enabled. Make sure your FreePBX server has the necessary modules installed and configured for WebSocket connections.
