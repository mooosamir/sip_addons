Easy Installation & Setup

Installation

  1. Download and extract the module to your addons directory
  2. Restart Odoo server
  3. Update Apps List
  4. Install "VoIP WebRTC FreePBX"
  5. Configure your FreePBX server connection

Configuration

  1. Navigate to VoIP → Servers
  2. Create new VoIP Server with your FreePBX details
  3. Add VoIP Users with SIP credentials
  4. Configure recording and auto-answer settings
  5. Start making calls from the systray icon!

System Requirements

Server Requirements

  • FreePBX or Asterisk server with WebSocket support
  • WebSocket module enabled (chan_sip or chan_pjsip)
  • HTTPS enabled on Odoo server (required for WebRTC)
  • Valid SSL certificate for secure connections

Client Requirements

  • Modern browser (Chrome, Firefox, Edge, Safari)
  • Microphone and speaker/headset
  • Stable internet connection
  • Microphone permissions granted to browser
